# YieldPlay SDK Documentation

## Welcome to YieldPlay SDK

YieldPlay SDK is the simplest way for game developers to add prize pools to any game using DeFi yield. Your game logic stays unchanged - we handle stakes, farming, and prize distribution.

### What You'll Find Here

* **Getting Started** - Quick setup and first integration
