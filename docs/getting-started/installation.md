# Installation

## Requirements

- Node.js 16+ or Python 3.8+
- API Key from [YieldPlay Dashboard](https://dashboard.yieldplay.io)

## JavaScript/TypeScript

```bash
npm install @yieldplay/sdk
```

Or with Yarn:

```bash
yarn add @yieldplay/sdk
```

## Python

```bash
pip install yieldplay-sdk
```

## Get API Key

1. Sign up at [dashboard.yieldplay.io](https://dashboard.yieldplay.io)
2. Create a new project
3. Copy your API key
4. Store in environment variable:

```bash
export YIELDPLAY_API_KEY="your_api_key_here"
```
